+-------------+
| DESCRIPTION |
+-------------+

With these BBcodes you can prevent smiley and other bbcode parsing.


+--------------+
| INSTALLATION |
+--------------+

1. Upload all files and folders to your ftp root;
2. Go to Admin -> System Administration -> BBCodes and 
3. just enable "!autonoparse", "noparse" and "tt" BBCodes, that's it!
